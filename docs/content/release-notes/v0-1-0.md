## What's in it

`hf` reads the Hugging Face Hub through the public REST API at
`huggingface.co/api`. No token or API key is required for public content.

### Commands

| command | what it does |
|---|---|
| `hf models` | Top downloaded models (`--task`, `--search`) |
| `hf model <id>` | Single model detail card |
| `hf datasets` | Top downloaded datasets (`--search`) |
| `hf dataset <id>` | Single dataset detail card |
| `hf spaces` | Top liked Spaces (`--search`) |
| `hf space <id>` | Single Space detail |
| `hf papers` | Daily papers feed (`--search`) |

All commands support `-o table|json|jsonl|csv|tsv|url|raw`, `--fields`,
`--template`, `-n/--limit`.

## Install

```bash
go install github.com/tamnd/huggingface-cli/cmd/hf@v0.1.0
```

Or download a prebuilt binary from the [releases page](https://github.com/tamnd/huggingface-cli/releases/tag/v0.1.0).
